# Projeto-DevCard-IA

Projeto desenvolvido para a disciplina de Aplicações Móveis (Aula 7). 

## Telas do App
1. **Boas-vindas**: Tela inicial de apresentação.
2. **Cadastro**: Formulário com validação de dados e seleção de cores.
3. **Preview**: Renderização dinâmica do cartão com lógica de badges por experiência.
4. **Sucesso**: Confirmação de finalização.

## Tecnologias
- React Native / Expo
- Expo Router (Navegação baseada em arquivos)
- TypeScript
- Flexbox para estilização
